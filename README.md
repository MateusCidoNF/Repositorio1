# Cronômetro em Python
# Esté o repositório inicial de um projeto simples e de cunho educativo sobre como utilizar GitHub, atrelado à uma progamação simples e objetiva com intuito de aprender a utilização da ferramenta. Sendo feita por um estudante de ADS.
## Funcionalidades
- Iniciar e pausar o cronômetro.
- Resetar o cronômetro.

## Como Usar
1. Clique em "Iniciar" para começar a contagem.
2. Clique em "Pausar" para interromper a contagem.
3. Clique em "Resetar" para zerar o cronômetro.
4. Clique em "Sair" para sair do cronômetro.
